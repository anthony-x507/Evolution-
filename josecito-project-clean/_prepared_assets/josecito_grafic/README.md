# Josecito/DIGOS Terminal Banner Asset

Prepared visual asset for the Josecito/DIGOS first-run terminal banner.
The approved static terminal version is wired through:

- `digos_lib/terminal_presentation.py`
- `digos_lib/onboarding.py`

## Files

- `josecito_grafic.png` - visual concept for the terminal welcome banner.
- `terminal_banner_spec.txt` - intended terminal layout and integration notes.

## Intended Use

The PNG remains the visual reference. The onboarding uses a static ANSI
interpretation so the experience works in ordinary terminals without extra
image dependencies.
